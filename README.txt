# Artificial-Life

## Running the app:
1. Install Go: https://go.dev/dl/
2. Run 'go mod tidy' to install dependencies
5. Execute with `go run ./cmd/main` (or compile with go build)

## Using the app:
1. To zoom in or out use the scroll wheel
2. Hold the left mouse button and drag to pan around
3. To start and stop use the space key, or right arrow key to move one step
4. Up or down arrow keys to speed up or slow down
